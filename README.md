# convertpdftojpg.in - Premium Free & Private Client-Side Converter

This is the complete source code for **convertpdftojpg.in**, a state-of-the-art client-side PDF-to-Image conversion utility. It is packed with premium, unique features and optimized for Google AdSense compliance and high search engine (SEO) rankings.

## Unique Features Added

1. **Page Rotation**: Users can rotate pages individually (90°, 180°, 270°) before converting, rendering the output correctly even for misaligned scanned documents.
2. **Selective Page Downloads**: Interactive custom checkboxes allow users to check/uncheck pages. The main download button dynamically changes to download only the chosen pages.
3. **Format Selection**: Converts PDF pages to standard **JPG**, lossless **PNG**, or modern **WebP** formats.
4. **Compression Quality Control**: A slider allowing compression selection (10% to 100%) so users can target file size parameters (e.g. "under 100kb" required by Indian government portals).
5. **Zero Server Footprint**: 100% of processing runs in the user's browser, giving you $0 hosting overheads and users absolute privacy.

## Project Structure

- `index.html` - Premium UI, options panels, select all utilities, and detailed security SEO FAQs.
- `app.js` - PDF.js render pipeline, dynamic state handlers, and ZIP packaging script.
- `styles.css` - Custom style parameters, grid animations, format button rules, and custom slider visuals.
- `about.html` - mission detail page.
- `contact.html` - user support contact portal.
- `privacy.html` - Google AdSense-compliant cookie & DoubleClick disclosures.
- `terms.html` - free-tier terms and disclaimers.

## How to Test and Run Locally

Since this is a client-side static application, you can run it easily:

### Option 1: Live Server (Recommended)
If you use VS Code, install the **Live Server** extension, right-click `index.html`, and select **Open with Live Server**.

### Option 2: Python HTTP Server
Open your terminal inside this folder and run:
```bash
python -m http.server 8000
```
Then navigate to `http://localhost:8000` in your web browser.

## Setup & Deployment Guide to Hit Millions of Traffic

To scale this domain and begin earning passive income, execute the following steps:

### Step 1: Deploy to a Free CDN (Hosting)
Because the converter runs entirely on the client side, your hosting costs will be $0. Choose one of these platforms to publish the files:
- **Netlify:** Drag and drop this folder directly into the Netlify dashboard.
- **Vercel:** Run `vercel` command or link a GitHub repository.
- **GitHub Pages / Cloudflare Pages:** Free, fast, global CDNs with automatic SSL certificates.

### Step 2: Configure custom domain on the host
Link your domain `convertpdftojpg.in` to the hosting provider by updating the CNAME/A records in your domain registrar (e.g., GoDaddy, Namecheap, Hostinger).

### Step 3: Google Search Console & XML Sitemap
1. Submit your site to [Google Search Console](https://search.google.com/search-console).
2. Create and submit an `sitemap.xml` file mapping these 5 pages to help Google index your site immediately.

### Step 4: AdSense Approval Checklist
Once your site is deployed and has been indexed for a few days:
1. Go to [Google AdSense](https://www.google.com/adsense) and click "Add Site".
2. Add your domain `convertpdftojpg.in`.
3. AdSense will provide you with a code snippet. Copy and paste it inside the `<head>` tag of all 5 HTML files.
4. Replace the `<div class="ad-slot">` placeholders with your actual AdSense ad units (`<ins class="adsbygoogle" ...>`).
5. Upload the generated `ads.txt` file to the root of this folder to prevent unauthorized ad injection.
